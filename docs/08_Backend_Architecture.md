# Backend Architecture

**Milestone:** Future (post Milestone 1)

## Overview

The KDVC Biometric Terminal will communicate with the KDVC backend over WiFi for fingerprint enrollment, verification, and dashboard integration.

This is intentionally out of scope for Milestone 1.

## Planned Components

| Component | Purpose |
|-----------|---------|
| REST API | Device-to-server communication |
| Database | Fingerprint template storage |
| Dashboard | Admin and reporting interface |
